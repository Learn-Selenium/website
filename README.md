Getting Started with Slate
------------------------------

### Prerequisites

You're going to need:

 - **Linux or OS X or Windows**.
 - **Ruby, version 2.2.5 or newer**
 - **Bundler** â€” If Ruby is already installed, but the `bundle` command doesn't work, just run `gem install bundler` in a terminal.

### Getting Set Up

1. Fork this repository on Github.
2. Clone the repo with `git clone https://github.com/Learn-Selenium/website.git`
3. `cd slate`
4. Initialize and start Slate.

```shell
# either run this to run locally
bundle install
bundle exec middleman server
```

You can now see the docs at http://localhost:4567. Whoa! That was fast!
